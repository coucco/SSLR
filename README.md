# Music Structured Controller

Проект для контролируемой генерации музыки с использованием Music VAE и структурированного латентного пространства.

## Установка

```bash
# 1. Клонировать проект
cd /path/to/project

# 2. Запустить установку
chmod +x scripts/setup.sh
./scripts/setup.sh

# 3. Активировать окружение
source venv/bin/activate
